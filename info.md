# Philips Heater CoAP Integration

A simplified Home Assistant integration for **Philips heaters** (CX3120, CX5120) using CoAP protocol.

## About

This is a **simplified rewrite** of the [philips-airpurifier-coap](https://github.com/kongo09/philips-airpurifier-coap) project, focused exclusively on heater functionality.

**Credit**: Original implementation by [@kongo09](https://github.com/kongo09)
